# Design Standards for Web Apps

All client-facing web applications:

- Dark mode by default with light mode option
- Color palette: Dark backgrounds (#0A0A0A, #1A1A1A), cyan accent (#00BCD4), white text (#FFFFFF), gray secondary text (#999999)
- Typography: Inter or similar modern sans-serif
- Lots of whitespace. Do not cram information.
- Mobile-first responsive design
- Smooth animations and transitions
- Loading states with progress indicators (scraping takes 2-5 minutes, show animated steps)
- The Level Up Academy branding throughout
- Premium SaaS feel. Think Linear.app or Notion aesthetics. Not generic Bootstrap.
