# The Tech Stack

## Data Storage
- **Airtable:** Central database. Single source of truth for all member data, reports, engagement, pipeline, and team members. Connected to everything via API.

## Data Collection (No credentials needed from salon owners. All public data.)
- **Apify:** Web scraping platform. Scrapes Instagram profiles, Facebook pages, websites, Google Business Profiles, and competitors using pre-built actors via API.
- **Search Atlas:** SEO intelligence platform. Keyword rankings, local search tracking, GBP performance, domain authority, AI search visibility. Enterprise plan for 15 websites. Used for Inner Circle and agency clients ONLY. NOT used per-member for Level Up. Level Up members get reports from Apify + Claude.
- **AI Accountant (hosted on Manus):** Financial analysis tool. Members input their numbers. Generates profit analysis, expense breakdowns, margin calculations.
- **Amplify:** Social media scraping and analysis tool. Used for Inner Circle social media strategy execution.

## Intelligence
- **Claude API (Anthropic):** The intelligence engine. Takes raw data from all sources and generates personalized blueprints, reports, and strategies using Nick's frameworks. ALL client-facing documents are generated by Claude.
- **Claude Code (Cursor):** Development environment where all automation scripts are built and run. Scripts can run on Nick's laptop or desktop (synced via GitHub).

## Client-Facing
- **Notion:** Customer portal. Each member gets a private Notion page as their business intelligence dashboard. Contains blueprints, monthly reports, action plans, and resources. Updated via Notion API automatically. Members access via private link, no Notion account needed.
- **Mighty Networks:** Community platform. Coaching calls, courses, Five Forces curriculum, Next Level Stylist training, Shopify e-commerce course. For community, content, and coaching ONLY. Not for reports or data delivery.

## Communication
- **Quo (formerly OpenPhone):** Business phone system. ALL client communication via text and calls. Welcome messages, report notifications, check-ins, at-risk outreach. PRIMARY communication channel.
- **Gmail:** Secondary. Used ONLY for onboarding welcome emails with login credentials and Notion portal links.

## Payments
- **Shopify Subscriptions:** Processes $2,000 + $750/month payments and $8,500 pay-in-full payments.

## Hosting
- **Manus:** Hosts the AI Accountant and V1 of new web apps (NPS tool, Visibility Analyzer). Good for rapid prototyping and initial deployment.
- **Cloudflare:** Production hosting for scaled applications. Nick has an existing Cloudflare account. Cloudflare Pages (frontend) + Workers (backend). Free tier handles most traffic.

## Automation
- **Make (Integromat):** Simple trigger automations ONLY. Typeform to Airtable. Status changes to notifications. NOT for intelligence work. Claude handles all intelligence.

## Lead Capture
- **Typeform:** 6-question qualification form. Drives leads into Airtable Pipeline table. Connected to Facebook ads and organic traffic.

## API Keys Required

Store in config/api_keys.env or .env file. NEVER commit to version control.

```
ANTHROPIC_API_KEY=           # Claude API
APIFY_API_TOKEN=             # Apify scraping
AIRTABLE_API_KEY=            # Airtable database
AIRTABLE_BASE_ID=            # Level Up Academy base
NOTION_API_KEY=              # Notion portal updates
NOTION_DATABASE_ID=          # Member portal database
ZOOM_ACCOUNT_ID=             # Zoom attendance
ZOOM_CLIENT_ID=              # Zoom OAuth
ZOOM_CLIENT_SECRET=          # Zoom OAuth
QUO_API_KEY=                 # Quo text messaging
SEARCH_ATLAS_API_KEY=        # Search Atlas (Inner Circle only)
GOOGLE_PAGESPEED_API_KEY=    # PageSpeed Insights
```
